# clean code

1. clean code adalah konsep penulisan kode untuk memudahkan pembaca agar lebih mudah di mengerti.

2. penulisan clean code harus deskriptif, konsisten (menggunakan camel case atau snake case), dan informatif.

3. menggunakan KISS atau keep it so simple. fungsi digunakan hanya untuk satu tugas A, tidak boleh untuk melakukan banyak tugas
